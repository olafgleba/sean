*Sean* is a frontend development environment to build Gulp driven projects.

## Getting started

* Clone the git repo: `$ git clone https://github.com/olafgleba/sean.git`
* `npm install`, `bower install`

## Development

* `$ gulp build`

## Production

* `$ gulp build --production`
